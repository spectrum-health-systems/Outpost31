Detailed Description

The Tingen configuration settings.

About this class
This class initializes the Tingen configuration settings, which:

  • Are specific to Tingen core functionality (other components/modules have their own configuration files)
  • Do not change between Tingen sessions
  • Can be modified by the user to suit their environment
  • Are stored in "TingenData\%SystemCode%\Config\Tingen.config"

More information
Configuring Tingen

string Outpost31.Core.Configuration.ConfigSettings.ModOpenIncidentMode
getset

Determines if the Open Incident Module functionality is enabled.

About this property

  • When enabled, the Open Incident Module will do work and (potentially) return a modified OptionObject.
  • When disabled,the Open Incident Module will not do work and will return an unmodified OptionObject.
  • These are the valid Open Incident Module modes:
    Mode Description
    Enabled The Open Incident Module is enabled (default).
    Disabled The Open Incident Module is disabled
