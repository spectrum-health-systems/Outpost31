Detailed Description

Avatar-specific data and logic.

This class should only contain the following Avatar-specific data:

  • The AvatarSystemCode
  • The SentScriptParameter
  • The SentOptionObject
  • The WorkOptionObject
  • The ReturnOptionObject
  • This object is added to the Tingen Session object when Tingen starts.

◆ SentScriptParameter

string Outpost31.Core.Avatar.AvatarData.SentScriptParameter
getset

The Script Parameter sent from Avatar.

The Script Parameter is a "-" delimited string with the following compoents:

Module The Tingen Module that will be doing the session work (ex: "Admin").
Command The Tingen Command request (ex: "All").
Action The Tingen Action request (ex: "Status").
Option The optional Action Option (ex: "Update").
  • More information about the Avatar Script Parameter here

.

The following Script Parameter will update all status files:

Admin-All-Status-Update
