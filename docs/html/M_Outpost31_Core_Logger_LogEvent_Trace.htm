      Remarks

      Trace logs are used to record information about the application's execution.

      • Records information about Tingen's execution
      • Used when debugging/troublshooting during development, and should probably be disabled in production
      • Important details are in the filename: %assemblyName%-%calledClass%-%calledMethod%-%lineNumber.trace
      • Do not contain any data
      • To ensure all logs are captured, filenames start with a timestamp: ssfffffff_
      • Extension is .trace
      - More information about Trace logs here.
