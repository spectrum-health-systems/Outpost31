Builds the Tingen Session object.

Parameters
configFileThe path to the Tingen configuration file.
sentOptionObjectThe OptionObject sent from Avatar.
sentScriptParameterThe ScriptParameter sent from Avatar.

The Tingen Session contains all of the information that Tingen needs to do what it does, including:

  • Configuration settings (user-definable settings from the configFile )
  • Static settings (these do not change between sessions)
  • Runtime settings (unique to the current session)
  • Data sent from Avatar, including the sentOptionObject and sentScriptParameter
  • Data derived from the sentOptionObject or sentScriptParameter .
  • Required Tingen Module details.

More information about this method:
This method is one of the most important in Tingen, as it builds the Tingen Session object.

The Session object is built in this order:

  1. A basic TingenSession object is initalized with the current date, time, configuration settings, and Avatar data.
  2. Framework information is added seperately.
  3. Trace log information is added seperately.
  4. Module objects are added seperately.
Returns
An Tingen Session object.
