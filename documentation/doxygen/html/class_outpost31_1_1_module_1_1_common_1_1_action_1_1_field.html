To ensure that any sensitive data is removed at the end of a session, the filePath should adhere to the following standards:

  • The filename should always start with the currentAvatarUser
  • The filePath should be in tnSession.TnFramework.TemporaryPath

Since the filePath is user-definable, we will verify that the filePath doesn't exist before (re)creating it.
