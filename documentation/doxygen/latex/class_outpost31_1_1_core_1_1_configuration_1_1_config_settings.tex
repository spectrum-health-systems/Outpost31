\doxysubsection{Detailed Description}
Tingen configuration settings. 

Tingen configuration settings\+: 
\begin{DoxyItemize}
\item Are specific to Tingen core functionality (other components/modules have their own configurations) 
\item Do not change between Tingen sessions 
\item Can be modified by the user to suit their environment 
\end{DoxyItemize}


\begin{DoxyItemize}
\item The Tingen configuration file is located in {\itshape "{}\+Tingen\+Data\textbackslash{}\%\+System\+Code\%\textbackslash{}\+Config\textbackslash{}\+Tingen.\+config"{}}.~\newline

\item More information about configuring Tingen here.
\end{DoxyItemize}
