\doxysubsection{Detailed Description}
System Code paths for Tingen. 


\begin{DoxyItemize}
\item The System Code path is where Tingen stores most of it\textquotesingle{}s important day-\/to-\/day data.
\item There is a separate System Code path for each \doxylink{namespace_outpost31_1_1_core_1_1_avatar}{Avatar} System Code. 
\end{DoxyItemize}
